# 🛡️ PosteGuard AI - MVP

### Sistema de Monitoramento Postural e Análise Termológica
Este projeto utiliza visão computacional para auxiliar na conscientização postural em tempo real, focando em análise de tensão cervical, lombar e de ombros.

---

## 🚀 Funcionalidades
* **Mapeamento Ósseo:** Rastreamento de pontos articulares via MediaPipe.
* **Ondas de Calor:** Simulação de pontos de calor (Trigger Points) baseada na posição corporal.
* **Relatórios:** Gráficos dinâmicos de tensão para análise de especialistas.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Interface:** Streamlit
* **IA/Visão Computacional:** MediaPipe e OpenCV
* **Análise de Dados:** Pandas e Numpy
